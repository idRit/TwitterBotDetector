1. Install Python 3
2. Install Pip
3. Install NodeJS
4. Install Git

5. Open a command prompt and enter these commands:
6. git clone git@github.com:idRit/TwitterBotDetector.git 
7. cd TwitterBotDetector-master
8. pip install venv
9. python -m venv env

10. env\Scripts\activate.bat
11. pip install -r requirements.txt
12. cd api\FakeProfilescraper
13. npm install

14. Setup is now complete, enter this commands to run:
15. npm run start
